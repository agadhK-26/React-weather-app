# Weather App

A simple weather application built with React, Vite, and Material UI. It uses the OpenWeatherMap API to fetch and display real-time weather information for any city.

## Features

* Search weather by city
* View current temperature
* Feels-like temperature
* Min & max temperature
* Humidity
* Wind speed
* Weather description

## Tech Stack

* React
* Vite
* Material UI
* OpenWeatherMap API

## Getting Started

```bash
npm install
npm run dev
```

Create a `.env` file in the project root:

```env
VITE_API_KEY=YOUR_OPENWEATHER_API_KEY
```
